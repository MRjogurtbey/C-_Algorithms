# C# Data Structures and Algorithms

Bu depo, Marcin Jamro'nun "C# Data Structures and Algorithms" kitabını çalışırken ve LeetCode üzerinden algoritma problemleri çözerken yazdığım kodları içermektedir.

## Hedeflerim
- C# diline ve Nesne Yönelimli Programlama (OOP) prensiplerine hakim olmak.
- Veri yapılarının (Arrays, Lists, Dictionaries, vb.) mantığını ve bellek yönetimini anlamak.
- Algoritma karmaşıklığını (Big O Notasyonu) analiz ederek optimize çözümler üretmek.

## Klasör Yapısı
- `/Arrays`: Dizi temelli problemler ve çözümler.
- `/Algorithms`: Temel algoritma mantığı ve çalışma prensipleri.
